# Lumi

Lumi is an experimental cognitive engine designed to move beyond pattern matching and static prompting.

Instead of acting as a traditional chatbot, Lumi attempts to construct an internal understanding of information, maintain contextual memory, form relationships between concepts, and adapt over time.

The project explores a simple question:

Can an AI understand information rather than merely predict text?

---

## Core Philosophy

Most language models operate as stateless prediction systems.

Input → Prediction → Output

Lumi introduces an intermediate layer:

Input
 ↓
Understanding
 ↓
Memory
 ↓
Reasoning
 ↓
Output

The goal is not to create another assistant.

The goal is to create a system capable of building an evolving internal world model.

## Vision

Lumi is intended to become a cognitive operating layer capable of:

- Understanding information
- Building memory structures
- Learning relationships
- Maintaining persistent context
- Reasoning over time

Rather than behaving like a chatbot, Lumi should behave more like a continuously evolving digital mind.

## Example

> lumi ask "What is recursion?"

Lumi should not simply repeat a definition.

Instead it should:

1. Understand the concept
2. Link it to previous knowledge
3. Build internal representations
4. Generate an explanation
